# 2025 Blog Public

这是我的个人博客项目，基于 Next.js、React、TypeScript 和 Tailwind CSS 开发。
## 声明

本项目基于开源博客模板进行二次开发，
已根据个人需求进行功能与界面修改。

## 功能

### 首页

- **可拖拽卡片布局** — 自由排列个人介绍、文章、图片、时钟、日历等卡片
- **个人介绍卡片** — 头像轨道动画、HyperText 文字动效、侧面板编辑个人简介
- **最新文章卡片** — 悬停展开显示封面和摘要，点击跳转全文
- **社交按钮** — 可配置的社交链接入口
- **站点配置面板** — 可视化编辑站点标题、主题色、卡片样式等

### 博客

- **Markdown 写作** — 内置编辑器，支持实时预览
- **代码高亮** — 基于 Shiki 的语法高亮
- **数学公式** — KaTeX 渲染
- **目录导航** — 自动生成文章 TOC 侧栏
- **本地保存** — 直接保存到本地文件系统，无需 GitHub 密钥

### 点赞系统

项目支持线上点赞（支持防刷限流，每天单 IP 限制一次）。点赞数据通过 Vercel KV 或 Upstash Redis 进行保存。
在 Vercel 部署时，请在环境变量中绑定 **Vercel KV**（自动注入 `KV_REST_API_URL` 和 `KV_REST_API_TOKEN`）。

### 图片展示

- **3D 图片轮播** — 透视旋转的立体卡片展示
- **记忆星图日历** — 按日期浏览图片，复古蓝+酒红配色
- **3D 翻转卡片** — 正面图片、背面描述，随机展示当天照片
- **Lens 透镜查看** — 点击图片打开详情，鼠标悬停放大细节
- **图片上传与管理** — 支持批量上传、描述编辑、删除

### 项目展示

- **项目卡片** — 展示项目名称、描述、标签、封面图
- **在线编辑** — 添加、编辑、删除项目，保存到本地

### 随笔

- **侧面板写作** — 轻量级笔记，从侧边滑出编辑

### 其他功能

- **友链管理** — 博主互链展示
- **音乐播放器** — 全局音乐卡片
- **图片压缩工具** — 内置图片体积优化
- **Live2D** — 虚拟形象展示
- **RSS 订阅** — 自动生成 RSS Feed
- **Sitemap** — 自动生成站点地图
- **主题定制** — CSS 变量驱动，全局可配色   
- **Cloudflare 部署** — 通过 OpenNext 适配 Cloudflare Workers
